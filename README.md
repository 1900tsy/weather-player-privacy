# weather-player-privacy
《天气玩家》隐私政策
生效日期： 2025年12月4日

【魏然】（以下简称“我们”）非常重视用户的隐私保护。本隐私政策说明了我们在您使用《天气玩家》（以下简称“本应用”）时如何收集、使用和保护您的信息。

一、我们要收集的信息及方式
1. 地理位置信息 (Location) 为了向您提供精准的当地天气实况及预报服务，本应用会申请访问您的地理位置权限（大概位置或精确位置）。

用途： 仅用于向气象服务商接口查询当前坐标的天气数据。

保护： 位置信息仅在您的设备本地使用，我们不会将您的行踪轨迹上传至我们的服务器进行存储或追踪。

2. 设备信息 (Device Information) 为了保证像素画风在不同设备上的显示效果，以及进行必要的应用崩溃排查，我们要收集您的非个人身份设备信息。

内容包括： 设备型号、操作系统版本、屏幕分辨率、系统语言设置。

用途： 仅用于应用适配和体验优化。

3. 不收集个人身份信息 本应用不需要注册账号即可使用。我们不会收集您的姓名、手机号、身份证号、照片或通讯录等个人敏感信息。

二、权限调用说明
除了地理位置权限外，本应用可能会申请以下权限以实现特定功能：

1. 通知权限 (Notifications)

用途： 用于发送极端天气预警，以及像素宠物（猫咪）的互动提醒（如深夜睡眠提醒、早安问候）。

控制： 您可以在系统设置中随时关闭此权限。

2. 网络访问权限 (Internet)

用途： 用于连接气象服务商服务器以更新最新的天气数据。

三、第三方服务声明
本应用依靠第三方服务提供核心气象数据。当您刷新天气时，必要的请求参数（如城市名称、经纬度坐标）将发送给服务商。

气象数据服务商： 和风天气 (QWeather)

他们的隐私政策：https://www.qweather.com/terms/privacy

四、数据存储与管理
1. 本地存储 您的所有个性化数据（包括收藏的城市列表、单位设置、已解锁的像素成就等）均存储在您设备的本地存储空间 (Local Storage) 中。我们无法在云端查看或备份这些数据。

2. 数据删除 由于数据存储在本地，如果您希望删除所有数据，只需卸载本应用或在系统设置中清除应用数据即可。数据一旦删除无法恢复。

五、儿童隐私保护
本应用致力于保护儿童隐私。我们不会有意收集或请求未满 13 周岁（或您所在司法管辖区规定的其他年龄）儿童的个人信息。如果您发现我们在不知情的情况下收集了儿童信息，请联系我们，我们将尽快删除。

六、隐私政策的更新
随着产品功能的迭代（例如增加新的游戏化功能），我们可能会更新本隐私政策。更新后的版本将通过应用内弹窗或公告的形式通知您。继续使用本应用即表示您同意受修订后的隐私政策约束。
